# Ask User Question

## Ziel

Stelle Philipp konkrete, leicht beantwortbare Entscheidungsfragen.

## Regeln

- Keine vagen Fragen.
- Maximal 1 bis 3 Fragen auf einmal.
- Jede Frage enthält Optionen.
- Gib eine Empfehlung mit kurzer Begründung.
- Frage nur, wenn die Antwort die Umsetzung relevant beeinflusst.

## Ausgabeformat

### Frage

[Konkrete Frage]

### Warum ich frage

[Kurze Begründung]

### Optionen

A) [Option A]  
B) [Option B]  
C) [Option C]  
D) Ich soll eine Empfehlung treffen

### Empfehlung

[Empfohlene Option und kurze Begründung]
