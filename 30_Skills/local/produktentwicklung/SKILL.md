# Produktentwicklung

## Ziel

Verwandle unstrukturierte Ideen in umsetzbare Produktkonzepte.

## Standardablauf

1. Idee zusammenfassen.
2. Zielgruppe und Problem klären.
3. Nutzenversprechen formulieren.
4. Features in Muss, Sollte und Später gruppieren.
5. MVP definieren.
6. Risiken und offene Fragen sammeln.
7. Nächste Umsetzungsschritte vorschlagen.

## Wenn Informationen fehlen

Nutze den Skill `ask-user-question`.

## Ausgabeformat

- Kurzbeschreibung
- Zielgruppe
- Problem
- Nutzenversprechen
- MVP-Features
- Spätere Features
- Risiken
- Offene Fragen
- Nächster sinnvoller Schritt
