# Skill Autoreview

## Ziel

Verbessere Skills, ohne sie unnötig aufzublähen.

## Prüfkriterien

- Ist der Trigger in der Description klar?
- Ist der Zweck klar?
- Ist der Output klar?
- Ist der Skill kurz genug?
- Gibt es unnötige Allgemeinplätze?
- Gibt es Widersprüche zu `AGENTS.md`?
- Gibt es konkrete Qualitätskriterien?
- Gibt es Sicherheits- oder Datenschutzrisiken?
- Sollte Inhalt in eine Referenzdatei ausgelagert werden?

## Ausgabeformat

- Bewertung
- Hauptprobleme
- Konkrete Verbesserungsvorschläge
- Überarbeitete Kurzversion
